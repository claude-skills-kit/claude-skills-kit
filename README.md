# claude-skills-kit

Claude Code plugin marketplace with custom skills.

## Install

```
/plugin marketplace add tanapoljoe1-art/claude-skills-kit
/plugin install fable-advisor@claude-skills-kit
```

## Plugins

- **fable-advisor** — Consult Fable 5 as a one-shot planning advisor while keeping quota usage low. Invoke with `/fable-advisor` (or say "ask fable" / "ถาม fable").
